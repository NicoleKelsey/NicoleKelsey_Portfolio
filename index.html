Timeline

Tools

Project Overview

Designing a secure, premium dealer-only web platform for a global luxury automotive manufacturer. The system needed to facilitate the procurement of specialized diagnostic tools, spare parts, and dealer licenses for authorized workshops globally.

The Challenge

The existing system was fragmented and didn't reflect the premium nature of the brand. I needed to create a unified experience that simplified complex ordering workflows while maintaining strict security and authorization protocols.

Impact & Outcome

Successfully delivered a design that unified three separate purchasing channels into one streamlined interface, reducing dealer support requests and improving order accuracy.

Container Yard Management System

Role
UX Designer
Timeline
1 Month
Industry

Client

Project Overview

A unified CYMS for automotive logistics—integrating real-time vehicle tracking, color-coded interactive yard maps, and mobile-first operator workflows into one data-driven dashboard.

The Approach

Focused on "Operator-First" design, I created a system that works as well on a rugged tablet in the yard as it does on a desktop in the office. Real-time data visualization was key to preventing bottlenecks.

Key Achievement

Reduced the time to locate and process vehicles by 35% through the implementation of a real-time, color-coded interactive yard grid.
